CHIP-8

Overview

CHIP-8 is an interpreted programming language that runs on a CHIP-8 virtual machine. It was developed in the 1970's to allow easily programmable games.

Technical Details

  • 4kB memory
  • 16 8-bit data registers
  • 64 byte stack
  • 64x32 monochrome display
  • 35 opcodes
  • 16 key hex keypad
